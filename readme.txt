an easy way to watch or build your sass file 
step-1: install sass global on your computer 
          npm install -g sass -> use this command and check the version by  running
           command sass --version . 
step-2:create css and sass file. open your terminal and run the following command
           sass --watch input.scss output.css ( type the exact directy ex. sass --watch sass/main.scss css/style.css)
